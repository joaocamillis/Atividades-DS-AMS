# Logcat Button - Atividade de PAM 2

Projeto Android desenvolvido em Kotlin utilizando Jetpack Compose para demonstrar a interação entre interface de usuário e o sistema de logs do Android (Logcat).

## Descrição do Projeto

O aplicativo apresenta um formulário simples onde o usuário insere seu nome e seleciona uma nota/menção acadêmica. Ao clicar em um dos botões, uma mensagem formatada é enviada para o Logcat com diferentes níveis de prioridade (Error, Warning, Debug e Info).
