Alguns Navegadores associados ao Desenvolvimento Web
Firefox Mozilla Firefox é um navegador livre e multiplataforma desenvolvido pela Mozilla Foundation com ajuda de centenas de colaboradores. A intenção da fundação é desenvolver um navegador leve, seguro, intuitivo e altamente extensível.
Chrome O Google Chrome é um navegador de internet desenvolvido pela Google.
Internet
Explorer		 Internet Explorer foi uma série de navegadores web gráficos desenvolvidos pela Microsoft e inclusos como parte da linha de sistemas operacionais Microsoft Windows, iniciado em 1995
Safari O Safari é um navegador de internet desenvolvido pela Apple Inc. Ele foi introduzido como o navegador padrão do sistema operacional Mac OS X v10.3 em 2003.
Netscape Netscape Navigator foi um navegador web proprietário que foi o principal produto da Netscape Communications Corporation.
