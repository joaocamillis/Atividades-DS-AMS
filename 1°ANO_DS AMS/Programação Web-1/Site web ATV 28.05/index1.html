Desenvolvedor Web é o termo utilizado para descrever o Desenvolvimento de sites, na
internet ou numa intranet.

O profissional que trabalha desenvolvendo websites pode ser um Web Designer
(Desenvolvedor do Layout), ou Web Developer (Desenvolvedor de Sistemas).

O Desenvolvedor refere-se a um processo de construção e testes de software especifico para a Web, com finalidade de se obter um conjunto de programas que satisfazem as funções pretendidas, quer em termo de usabilidade dos usuarios ou compatiblidade com os programas existentes

O Desenvolvedor Web pode variar desde simples paginas estaticas a aplicações ricas, comércios eletrônicos ou redes sociais.
